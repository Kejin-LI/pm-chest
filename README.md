# PM Chest (产品经理百宝箱)

这是一个 Trae 自定义 Skill，专为产品经理和独立开发者设计。它可以帮助你全流程自动化地完成产品设计工作。

## 核心功能

1.  **需求澄清**：像资深 PM 一样通过反问帮你梳理模糊的想法。
2.  **PRD 撰写**：生成标准化的产品需求文档，包含背景、流程、详细交互表和数据埋点。
3.  **UI 设计与原型**：提供多种风格选择，生成高保真的 HTML/CSS/JS 交互原型。

## 如何安装

1.  找到你的 Trae 项目根目录。
2.  进入 `.trae/skills/` 文件夹（如果没有，请手动创建）。
3.  将本文件夹 `pm-chest` 完整复制到 `.trae/skills/` 下。
4.  重启 Trae 或重新加载窗口。

## 如何使用

在对话框中输入以下任意指令即可触发：
-   "启动 PM Chest"
-   "帮我写一个背单词小程序的 PRD"
-   "我想做一个智能家居控制中心"
